BMI Calculator



BMI stands for Body Mass Index, which is a measure of body fat based on a person's weight and height. It is a commonly used tool to assess whether an individual's weight is within a healthy range or not. BMI is calculated by dividing a person's weight (in kilograms) by their height (in meters) squared. BMI can be used to categorize a person's weight as underweight, normal weight, overweight, or obese.

There are several levels of BMI Index:

  • < 18.5 Underweight
  • 18.5 - 24.9 Normal weight
  • 25.0 - 29.9 Pre-obesity
  • 30.0 - 34.9 Obesity class I
  • 35.0 - 39.9 Obesity class II
  • > 40 Obesity class III
